<h4>Some useful thing to work with saigonparking environment <br/></h4>

- Start docker compose:
    <br/>&emsp;&emsp; $ docker-compose up

- Stop docker compose:
    <br/>&emsp;&emsp; Ctrl + C
    
- Remove all containers:
    <br/>&emsp;&emsp; $ docker-compose rm
