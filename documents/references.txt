116. Does Java pass by reference or pass by value?
        https://www.infoworld.com/article/3512039/does-java-pass-by-reference-or-pass-by-value.html
    ==> "Passing by value" refers to passing a copy of the value
    ==> "Passing by reference" refers to passing the real reference of the variable in memory
    ==> A mutable object’s value can be changed when it is passed to a method, such as Builder object
    ==> An immutable object’s value cannot be changed, even if it is passed a new value, such as primitive type
